## üöÄ **GETTING STARTED WITH ENHANCED BMAD**

### Prerequisites
- Node.js >= 16.0.0
- npm >= 8.0.0
- GitHub Token with repository access

### Installation
```bash
# Clone the enhanced repository
git clone https://github.com/helton-godoy/bmad-github-native-full-cycle.git
cd bmad-github-native-full-cycle

# Install enhanced dependencies
npm install

# Setup enhanced environment
npm run setup

# Verify enhanced installation
npm run status
```

### Configuration
```bash
# Copy enhanced environment template
cp .env.example .env

# Configure enhanced settings
GITHUB_TOKEN=your_github_token
GITHUB_OWNER=your_username
GITHUB_REPO=your_repository
BMAD_ENHANCED_MODE=true
BMAD_QUALITY_GATES=true
BMAD_PERFORMANCE_MONITORING=true
```
